# frog_runetlex

## ⚙️ Как установить и запустить

Так как API-ключ скрыт в целях безопасности, для запуска программы вам понадобится собственный ключ.

1. **Скачайте проект:**
   Нажмите зеленую кнопку **`<> Code`** в правом верхнем углу страницы -> выберите **`Download ZIP`**. Распакуйте скачанный архив в удобную папку.

2. **Настройте API-ключ:**
   * Зайдите в распакованную папку.
   * Найдите файл `.env.example` и переименуйте его в `.env` (просто сотрите `.example`).
   * Откройте этот файл `.env` в Блокноте или любом другом текстовом редакторе.
   * Вставьте ваш реальный API-ключ вместо текста-заглушки и сохраните файл.

3. **Запустите программу:**
   Убедитесь, что на вашем компьютере установлен Python. 
   Дважды кликните по файлу `run_app.bat` (или `run_app_chrome.bat`, если хотите, чтобы программа сразу открылась в браузере Chrome).
